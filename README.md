# Chess Engine with AI

## Overview
A fully functional chess engine built in Python using object-oriented programming principles. The application supports both two-player gameplay and an AI opponent.

## Features
- Complete legal move validation
- Check and checkmate detection
- AI opponent logic
- Two-player mode
- Object-oriented board and piece management

## Technologies Used
- Python
- Object-Oriented Programming

## How to Run
-Linux/macOS:
-source venv/bin/activate

-Windows:
-venv\Scripts\activate

-pip install pygame
-pip install pygame_gui

-python chess_gui_small_view.py
